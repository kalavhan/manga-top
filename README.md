 - Yarn
 - Node.js

 - PostgreSQL


## Getting Started

- git clone https://github.com/kalavhan/manga-top.git

- cd manga-top

- bundle update

- bundle install

- rails db:create

- rails db:migrate (to add tables and columns)

- rails db:seed (to add the categories)

- rails server to fire the rails server
